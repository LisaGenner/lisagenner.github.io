Native Chat App

This app was built for mobile devices using React Native. It provides users with a chat interface, options to share images and their location. I used Expo to develop the app. It is optimized for both Android and iOS devices. The chat interface and functionallity uses the Gifted Chat library. To store the chat messages, I used the Google Firestore Database. Images are stored in Firebase Cloud Storage. To make the chat conversations available offline , I used asyncStorage to store them locally. I used Google Firebase authentication to authenticate users anonymously.
